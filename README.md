# Shoesphere 👟

Shoesphere is a basic e-commerce website for buying shoes online.
This project is created using HTML, CSS, and JavaScript.

## Features

* Login page
* Homepage
* Shop page
* Buy Now page
* Contact page
* About page

## Technologies Used

* HTML
* CSS
* JavaScript

## Note

This is a front-end project and is not fully functional yet.
Backend, database, and payment features will be added in future.
